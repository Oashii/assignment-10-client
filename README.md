# PlateShare

PlateShare is a full-stack MERN application that allows users to share surplus food with the community, helping reduce food waste. Users can post food items they want to donate, browse available items, and request them.

---

## Features

- User authentication (Login/Register)
- Post and manage food items
- Browse and request food items from other users
- Manage your own donations and requests
- Responsive UI

---

## Technologies Used

- **Frontend:** React.js, React Router, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** Firebase
- **State Management & Data Fetching:** React Context, React Query
- **Other:** Axios for HTTP requests
